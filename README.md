# Student Management System

A Python desktop application for managing student records. Built for small schools and coaching centers.

## Features

- Add new students with name and marks
- View all students with grades (A, B, C, Fail)
- Search students by name
- Delete student records
- Update marks
- Show topper student
- Data automatically saved to file
- Dark theme UI
